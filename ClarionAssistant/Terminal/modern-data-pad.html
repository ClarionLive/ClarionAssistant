Power features of the CA / Monaco editor. A few chords are CA-specific and differ
                from stock Monaco — those are called out in the note at the bottom.


            
            

                
                ×
            


            
            

                
🧭Navigation & code intelligence

                

                    F12
                    Go to definition (at the text cursor)

                

                    Ctrl+Click
                    Go to definition (at the mouse pointer)

                

                    Alt+←orAlt+→
                    Navigate back / forward through jumps

                

                    Ctrl+Space
                    Trigger code completion

                

                    Hover
                    Hover a symbol for type info, signature & docs

            


            
            

                
🔎Find & replace

                

                    Ctrl+F
                    Open Find

                

                    Ctrl+H
                    Open Find & Replace

                

                    F3orShift+F3
                    Jump to next / previous match

                

                    EnterorShift+Enter
                    In the find box: next / previous match

                

                    Esc
                    Close the find panel

            


            
            

                
🖱️Multi-cursor & selection

                

                    Alt+Click
                    Add a cursor where you click

                

                    Ctrl+Alt+↑or↓
                    Add a cursor above / below

                

                    Shift+Alt+drag
                    Column (box) selection

                

                    Double-click
                    Select the word — Clarion-aware (Add:Field counts as one word)

            


            
            

                
✏️Line editing

                

                    Alt+↑or↓
                    Move the line up / down

                

                    Shift+Alt+↑or↓
                    Copy the line up / down

                

                    Ctrl+2
                    Duplicate the line above

                

                    Ctrl+Shift+Y
                    Remove the line

                

                    Ctrl+Shift+K
                    Delete the line

                

                    Ctrl+X
                    Cut — or clear the whole line if nothing is selected

                

                    TaborShift+Tab
                    Indent / outdent

            


            
            

                
💬Comments & case

                

                    Ctrl+/
                    Comment the line

                

                    Ctrl+Shift+/
                    Uncomment the line

                

                    Ctrl+Shift+U
                    Uppercase the selection

                

                    Ctrl+Shift+L
                    Lowercase the selection

                

                    Ctrl+\
                    Invert the case of the selection

            


            
            

                
🔖Bookmarks

                

                    Ctrl+F2
                    Toggle a bookmark on the current line

                

                    F2orShift+F2
                    Jump to next / previous bookmark

                

                    Gutter click
                    Click the left glyph margin to toggle a bookmark

            


            
            

                
🛠️Clarion tools

                

                    Ctrl+D
                    Open the Structure Designer for the structure at the cursor

            


            
            

                
👁️View & editor

                

                    Ctrl+Wheel
                    Zoom the editor font in / out

                

                    F1orCtrl+Shift+P
                    Open the command palette

                

                    Ctrl+S
                    Save

                

                    Fold
                    Use the chevrons between the line numbers and code to fold / unfold

CA-specific: a few chords differ from stock Monaco — Ctrl+D
                opens the Structure Designer (not multi-cursor), Ctrl+Click goes to
                definition (use Alt+Click to add a cursor), and
                F2 is Next Bookmark. All editor shortcuts are rebindable in the gear (⚙) panel's Keyboard section.
