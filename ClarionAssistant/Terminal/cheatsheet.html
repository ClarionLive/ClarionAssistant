Clarion Assistant Cheat Sheet

Things you can ask the assistant to do in the terminal
📄 Editor & Navigation
"Open MyFile.clw"Opens a file in the IDE editor
"Go to line 42"Navigates the cursor to a specific line
"Find all uses of MyVar"Searches for text in the active file
"Comment out lines 10-20"Toggles Clarion line comments
"Replace OldName with NewName"Find & replace in the active editor
🛠 Clarion Classes
"Analyze MyClass.inc"Parse class structure — methods, properties, data members
"Sync check MyClass"Find missing or orphaned method implementations
"Generate stubs for MyClass"Create .clw stubs for unimplemented methods
"Write a class that does X"Generates .inc and .clw from a description
🗂 Application & Embeditor
"List procedures"Lists all procedures in the open .app file
"Open the embeditor for Main"Opens the embed editor for a procedure
"Show me procedure details for X"Template, prototype, module, parent info
"Export the app to TXA"Exports the application as text for analysis
"Evaluate code"Analyze app code for issues & improvements
🔍 CodeGraph & Search
"Index this solution"Indexes all symbols & relationships in the solution
"Who calls UpdateOrder?"Traces callers across the entire codebase
"What does Main call?"Shows all outgoing calls from a procedure
"Find dead code"Finds symbols with no incoming references
"If I change X, what breaks?"Impact analysis across the solution
"Show the class hierarchy"Traces inheritance relationships
🎓 LSP — Language Server
"Where is X defined?"Go-to-definition across files
"Find all references to X"Every usage of a symbol in the workspace
"What type is X?"Hover info — type, signature, docs
"What symbols are in this file?"Lists procedures, classes, and variables
📚 Documentation Search
"How do I parse CSV with StringTheory?"Searches installed template docs
"Show me NetTalk email methods"Finds methods, params, and examples
"What FM3 methods handle backups?"Searches across all ingested libraries
"Ingest docs"Indexes docs from your Clarion install (run once)
⚖ COM Controls
"Create a COM control"Scaffolds a C# COM project for Clarion
"Build the COM project"MSBuild compile with manifest generation
"Validate the COM control"Checks RegFree COM compliance
"Create a WebView2 COM control"HTML/CSS/JS rendered control for Clarion
Tip: The assistant can read and edit files directly in your IDE. Just describe what you want to do — it has full access to your editor, app tree, solution index, docs, and language server.
